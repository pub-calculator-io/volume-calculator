The online volume calculator performs calculations for 11 different geometric shapes. The tool supports different units of measurement and demonstrates the solution steps.

[https://www.calculator.io/volume-calculator/](https://www.calculator.io/volume-calculator/)

== Usage ==

Add the Volume Calculator shortcode to your page, post or sidebar:

`[ci_volume_calculator]`

Add the following code to your template where you would like the Volume Calculator to appear:

`<?php display_ci_volume_calculator(); ?>`

== Installation ==

1. Upload the Volume Calculator /ci_volume_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Volume Calculator plugin through the "Plugins" menu in WordPress.
